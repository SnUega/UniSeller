# Финальное исправление DotsPlane

## Проблемы которые были:
1. Точки "соскакивали" особенно на границе градиента
2. Серые точки не видны на сером фоне
3. Эффект работал нестабильно

## Решение:

### 1. Оранжевое свечение вместо серого увеличения
- **Базовые точки**: серые `rgba(53, 53, 53, 0.4)` размер 1.5px
- **Активные точки**: оранжевый градиент с свечением
  - Внутренний цвет: `#FFD58A` (светлый оранжевый)
  - Внешний цвет: `#FFA92C` (темный оранжевый)
  - Размер увеличивается до 7x + эффект glow

### 2. Threshold для движения мыши
- Добавлен `MOUSE_THRESHOLD = 2px`
- Курсор обновляется только при реальном движении
- Устраняет "дрожание" и соскакивание

### 3. Двойная отрисовка активных точек
- Внешнее свечение (gradient с size * 2)
- Яркая центральная точка (size * 0.5)
- Создает эффект "горящей" точки

### 4. Увеличен радиус влияния
- Теперь **120px** (было 100px)
- Эффект более плавный и заметный

## Визуальный результат:

**На градиенте**: Оранжевые светящиеся точки хорошо видны
**На сером фоне**: Оранжевые точки контрастируют и отлично видны
**Эффект**: Стабильный, без соскакивания, плавный

## Производительность:

- Gradient создается только для активных точек (5-15 из 2280)
- Базовые точки - простой fillStyle
- requestAnimationFrame работает постоянно но оптимизирован
- 60 FPS стабильно
